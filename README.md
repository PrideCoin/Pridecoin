# Pridecoin

Pridecoin is a project aiming to develop and implement the world’s first all-in-one next generation decentralized social network platform that advocates for LGBTQ rights.
The project is based on the Ethereum blockchain and implements the PrideCoin token (XPR) as its own cryptocurrency and the PrideCoin ICO smart contract.
All the source code for the smart contracts is provided in this repository.
